## Sales & Revenue Analysis Dashboard (Tableau)
# Project Overview

This project presents an interactive Sales & Revenue Analysis Dashboard built using Tableau. The dashboard provides insights into sales performance, profit trends, regional distribution, and top-performing products.

# Objectives
Analyze overall sales and profit
Identify sales trends over time
Compare category performance
Evaluate regional contribution
Highlight top revenue-generating products
# Dashboard Features
1. KPI Metrics
Total Sales
Total Profit
2. Sales Trend Analysis
Monthly sales trend visualization
Identifies peak and low-performing periods
3. Category Performance
Comparison of sales across categories
Helps identify best-performing category
4. Sales by Region
Distribution of sales across regions
Shows regional contribution
5. Top Products
Displays highest revenue-generating products
# Filters
Region filter (applies to entire dashboard)
# Tools Used
Tableau (Data Visualization)
Dataset: Superstore Sales (Kaggle)
# Key Insights
Sales show fluctuations across months with peaks in later months
Technology category generates highest revenue
Certain regions contribute more significantly to overall sales
Few products dominate revenue generation
# Dashboard Design Principles
Minimal and consistent color usage
Clear layout with KPI at top
Proper alignment using containers
Interactive filtering
# How to Use
Open Tableau file (.twbx)
Interact with Region filter
Hover on charts for detailed insights
Analyze trends and comparisons
